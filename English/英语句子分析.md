```
分析句子：意义大于形式
先排除干扰，去掉状语；
分析句子成分（找到五大基本句型之一），句子结构完整即可立即翻译；
最后加上状语或修饰语，准确定位修饰的对象很重要；
非谓语最后分析，缺什么补什么；
分析从句也类似
```
```
宾语分为两类：动词宾语、介词宾语
定语分为：前置定语和后置定语

充当状语的成分：修饰动词、形容词、副词及整个句子
1. adv
2. 介词 + 宾语
3. 时间词
4. 表频率和倍数的词
5. 状语从句
```
```
“介词+名词”视作修饰语；
名词后面是定语，其他的是状语或补足语；
动词 + 名词 + “介词+名词”充当状语
```

```
A of B 看作一个大名词
```

```
名词词组：限定词1/限定词2/... + adv1/adv2/... + adj1/adj2/... + n1/n2/n3/...
```

```
系动词

（这里的动词为“是”动词）
表状态：be动词
表感官：look, sound, taste, smell, feel
表变化：get, become, turn, grow, fall
表保持：keep, stay, remain, stand
表表象：seem, appear
```

```
宾语补语（S + V + O + C）的句型也有一个很简单的检验方式：把宾语C和补语O拿出来、中间加个 be 动词，如果可以构造出一个 S + V + C 的句型而且意思差不多，那么它就是宾语补语的句型，即宾语和宾语补足语两者直接有关系。

1. S + V + A as(as可省略) B（将as B 看作宾语补足语）
2. 使动词：其后如果加宾语则必须有宾补。make/keep/consider/
```

```
表语无所不包；
非谓语可以充当任何句子成分，包括谓语；
非谓语动词和动词一样可以带宾语、副词、补语；不定式(to do)、分词(ving/done)、动名词(ving)
```

```
状语用来修饰动词、形容词、副词以及全句的句子成分。
根据作用分为（9种）：时间、地点、原因、目的、结果、条件、让步、程度和伴随；
状语的一个难点是找到其修饰的对象；

5种基本句子类型；
	S + V
	S + V + O 
	S + V + O + O 动词有两个对象，给某人某物
	S + V + O + C 
	S + V + C
7种句子成分：主语、谓语、宾语、补语（主语补语、宾语补语）、修饰语（定语、状语、同位语）
10种词类：
	实词：名词、代词、形容词、副词、动词、数词
	虚词：介词、冠词、连词、语气词
```

```
名词后面的从句可能是：定语从句、同位语从句，引导词充当句子成分的是定语从句。
名词后面的非谓语：通常作后置定语（修饰前面的名词）或状语
```

```
被动语态的分析：先转为主动语态
```

```
介词：
简单介词：in,on,at,under,above,to,over,off,about,for,from,of,with,without,after,by,down,before,into,through,within,till,during,between,out,away等
介词短语：这类短语的最后一个词一定是介词；
现在分词或过去分词形式的介词；
介词的宾语通常由名词或代词充当，若需动词作宾语，则通常用动名词形式；
介词短语的句法功能：
1. 作定语；
1. 作状语；
1. 作表语；
1. 作补语；
1. 作介词的宾语；
```
